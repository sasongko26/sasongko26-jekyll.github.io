Echo merupakan perintah bahasa bash (bahasa yang digunakan terminal) untuk menampilkan  output ke monitor.
```
$ echo “teks yang diinginkan”
teks yang diinginkan
```
Contoh:
```
$ echo "halloooo....." 
halloooo..... 
$ echo "BlankOn Linux Indonesia" 
BlankOn Linux Indonesia
```
